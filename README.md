# BannerHub for ReVanced

ReVanced patches for [GameHub](https://www.gamehubglobal.com/) (`com.xiaoji.egggame`) that add BannerHub features: community configs, frontend export (Beacon + ES-DE), in-app download manager, and more.

## Usage

Add this repository as a patch source in [ReVanced Manager](https://github.com/ReVanced/revanced-manager) or use directly with [ReVanced CLI](https://github.com/ReVanced/revanced-cli).
